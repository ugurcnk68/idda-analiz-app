
İdda Kupon Üretici - iPhone için (statik web uygulaması)

Bu paket içinde:
- index.html  -> ana uygulama
- app.js      -> hesaplama + UI mantığı (client-side)
- style.css   -> basit stil

İndirip iPhone'da açma adımları:
1) Bu zip'i indirip iPhone Files uygulamasına veya herhangi bir sunucuya yükleyin.
2) Eğer bir sunucuya yüklerseniz (ör. GitHub Pages, Netlify), doğrudan URL'yi Safari'de açın.
   Daha sonra Safari menüsünden "Add to Home Screen" (Ana Ekrana Ekle) seçin — uygulama gibi çalışır.
3) Eğer dosyayı doğrudan iPhone'a yüklediyseniz, Files uygulamasından index.html'e dokunup açmayı deneyin.
   (Not: Service Worker veya offline PWA özellikleri için HTTPS gerekir; local dosya olarak açınca bazı kısıtlar olabilir.)
4) Kullanım: Maç ekle, odds gir, "Analiz Et" ile banko ve value bahisleri görün. JSON olarak dışa aktarabilirsiniz.

Sınırlamalar:
- Uygulama canlı bookmaker API'siyle bağlanmaz; tüm veri kullanıcı tarafından girilmelidir.
- Daha gelişmiş, API destekli bir iOS uygulaması için sunucu, API anahtarları ve/veya App Store yüklemesi gerekir.

Sorumluluk: Bahis risklidir. Bu uygulama yalnızca bilgilendirme amaçlıdır.
